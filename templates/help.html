How to Use This Tool

This guide explains the main features of the JSON Webhook Server application.

Generate New Webhook

Clicking this button on the dashboard instantly creates a new, unique URL and a corresponding Webhook ID. The new URL is displayed in an alert box, ready for you to copy and use in your client applications to send data.

Check Received Payloads

This button appears next to a newly generated webhook. Clicking it takes you to a detailed page that lists all JSON payloads sent to that specific webhook. Payloads are displayed with the most recent at the top, along with their timestamps. You can download each payload individually.

View All Webhooks

Located on the dashboard, this button navigates you to a comprehensive list of every webhook ever generated by the application. From this list, you can see high-level stats and click through to view the detailed payload data for any webhook.
